Im Menü „Aktuelle Karte“ können Sie sich alle Teilnehmer auf einer Karte anzeigen lassen. Die Teilnehmer werden hierbei auf der Karte an Ihrem im Teilnehmerbreich vermerkten Wohnort dargestellt.
