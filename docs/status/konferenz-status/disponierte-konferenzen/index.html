Disponierte Konferenzen

Konferenzen, die nicht sofort gestartet werden, sondern mit einem Auslösezeitpunkt definiert wurden, sind im Menü „disponierte Konferenzen“ einsehbar. Disponierte Konferenzen können mit einem Klick auf das Löschsymbol gelöscht, jedoch nicht bearbeitet werden.
