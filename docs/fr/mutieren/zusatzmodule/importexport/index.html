Import/Export

Import/Export

L’import permet d’importer, outre les données des participants, les champs optionnels, les unités et leur structure hiérarchique. L’import est principalement conçu pour un «premier import» manuel.

Actualiser via l’import: chaque ligne de données comprend un code unique (ID) qui ne peut être attribué qu’une fois à chaque participant. Si l’ID se trouve déjà dans la base de données existante, les données correspondantes sont écrasées par les nouvelles; une mise à jour a donc lieu.

La procédure lors d’un “premier import” est décrite ci-après; elle reste toutefois la même pour les mises à jour ou les “imports ultérieurs”.
